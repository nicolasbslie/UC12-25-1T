O que é o HTML

A respeito de linguagem de marcação

HTML, sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), é a linguagem base utilizada para estruturar e organizar páginas da web.
Ela utiliza "tags" para definir elementos como títulos, parágrafos, links e imagens, agindo como o esqueleto de um site, não sendo uma linguagem de programação.
